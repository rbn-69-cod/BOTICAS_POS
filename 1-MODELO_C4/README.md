# Modelo C4 - MediZano

## Descripción

El modelo C4 permite representar la arquitectura de MediZano en distintos niveles de detalle. Esta documentación utiliza tres niveles principales:

| Nivel | Descripción |
|---|---|
| Nivel 1 - Contexto | Muestra MediZano como sistema principal, sus actores y sistemas externos. |
| Nivel 2 - Contenedores | Muestra las aplicaciones y servicios principales: frontend, backend, base de datos y servicios de soporte. |
| Nivel 3 - Componentes | Muestra la estructura interna del backend Spring Boot. |

## Objetivo

Documentar la arquitectura de MediZano de forma clara, permitiendo comprender cómo interactúan los usuarios, la interfaz web, la API REST, la base de datos y los servicios internos del sistema.
